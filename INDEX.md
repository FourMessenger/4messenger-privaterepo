## 🚀 HOW TO GET STARTED

### Installation
```bash
# Dependencies already installed
# Just start using the theme system!
npm run dev     # Development mode
npm run build   # Production build
```

### Using a Theme
```
Settings → Themes → Import Theme → Select .4mth file
```

### Creating a Theme
```
1. Create manifest.json
2. Create styles.css
3. ZIP together
4. Rename to .4mth
5. Test and share!
```
